# Api rest full

---
Estableciendo buenas prácticas en una api rest de spring boot

Contenido

Operaciones crud
Paginación
Payload usando genericos
Configuración de perfiles

### Comandos útliles

Limpiar target

```cmd
mvn clean
```

Compilar por defecto
```cmd
mvn install
```


Compilar perfil
```cmd
mvn install -Pdev
```
